정수형 (Integer Types):
byte - 크기: 1 byte
short - 크기: 2 bytes
int - 크기: 4 bytes

실수형 (Floating-Point Types):
float - 크기: 4 bytes

문자형 (Character Type):
char - 크기: 2 bytes

논리형 (Boolean Type):
boolean - 크기: JVM에 따라 다르지만 보통 1 byte
